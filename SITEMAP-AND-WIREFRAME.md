## 2. Wireframe Layouts

### Home Page (/)
| Section | Layout Notes |
|---------|-------------|
| **Header** | Logo left, nav center, "Order Now" button right |
| **Hero** | Full-width background image (farm sunrise). H1 headline + subheadline + CTA button |
| **Value Props** | 3-column grid: icons + short text (Farm Fresh / No Additives / Sustainable) |
| **Featured Products** | 3-card horizontal row with product image, name, short description, "Learn More" link |
| **Farm Story Teaser** | Split layout: image left, text right. Brief story + "Read Our Story" link |
| **Testimonials** | 2–3 customer quotes in a centered carousel or grid |
| **Newsletter CTA** | Centered section with email input + "Subscribe" button |
| **Footer** | 4-column: Brand info, Quick Links, Contact, Social icons |

### About Us (/about)
| Section | Layout Notes |
|---------|-------------|
| **Hero** | Half-height image banner with overlay text |
| **Our Story** | Alternating image-text blocks (left-right-left) telling the farm-to-table journey |
| **Our Farmers** | Photo grid or cards with short bios |
| **Quality Promise** | Icon list with commitments (no hormones, grass-fed, tested daily) |
| **CTA** | Full-width banner: "Taste the Difference" → Shop |

### Products (/products)
| Section | Layout Notes |
|---------|-------------|
| **Hero** | Short banner with headline |
| **Product Cards** | 3-column grid. Each card: large image, product name, description, nutrition highlight, "Order" button |
| **Why Choose Us** | Horizontal icon bar with differentiators |
| **CTA** | "Start Your Subscription" banner |

### Sustainability (/sustainability)
| Section | Layout Notes |
|---------|-------------|
| **Hero** | Nature-themed banner |
| **Practices** | Alternating image-text blocks (solar panels, happy cows, recyclable packaging) |
| **Impact Numbers** | 3-stat counter row (tons CO₂ saved, acres preserved, families supported) |
| **CTA** | "Support Sustainable Dairy" → Shop |

### Shop / Order (/shop)
| Section | Layout Notes |
|---------|-------------|
| **Header area** | Filter/sort bar |
| **Product Grid** | Cards with image, name, size options, price, "Add to Cart" |
| **Subscription Box** | Highlighted section: "Subscribe & Save 15%" with plan options |
| **Trust Badges** | Secure checkout, free shipping threshold, satisfaction guarantee |
